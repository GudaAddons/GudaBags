# GudaBags

A clean bag addon for WoW Anniversary Classic. Organize your items with categories, sort with one click, and use special bars for quest items and farming.

---

## Quick Access Bars

GudaBags has two floating bars that make your life easier:

### Quest Item Bar

![Quest Item Bar](SCREENSHOT_QUEST_BAR)

All your usable quest items in one place. No more searching through bags!

- See all quest items at a glance
- Click to use them directly
- Hover to see more quest items

### Tracked Item Bar

![Tracked Item Bar](SCREENSHOT_TRACKED_BAR)

Perfect for farming! Pin items you want to track and see the total count.

- **Ctrl+Alt+Click** any item to track or untrack
- Shows combined count from all bags
- Great for herbs, ores, cloth, reputation items

## Features

### Category View
Items are sorted into groups automatically. Easy to find what you need.

- Equipment, Consumables, Trade Goods, Quest Items, and more
- Drag items to move them to different categories
- Click category header to collapse/expand
- "Recent" category shows items you just picked up

### Single View
Classic bag layout. All items in one window, clean and simple.

### Smart Sorting
One click to organize everything:
- Hearthstone stays in first slot
- Items grouped by type
- Stacks combined together
- Junk moved to the end

### Bank Support
Same features for your bank. Category view or single view - your choice.

### Search
Type to find items quickly. Search by name or type.

### Multi-Character View
See other characters' bags and banks without logging in.

### Realm-Wide Tracking
- **Tooltip item count** - Hover over any item to see how many you have across all characters on the realm
- **Gold display** - See your total gold across all realm characters

## Customization

Open settings with `/gb config` or click the gear icon.

**What you can change:**
- Icon size (small to large)
- Number of columns
- Quality borders (colored by rarity)
- Junk highlighting
- Background opacity
- Show/hide quest bar
- Show/hide tracked bar
- Recent items time (how long items stay in "Recent")
- And more...

### Custom Categories
Create your own categories! Set rules to match items by:
- Item type (Armor, Weapon, etc.)
- Item name
- Item quality
- And more...

## Slash Commands

| Command | What it does |
|---------|--------------|
| `/gb` | Open/close bags |
| `/gb bank` | Open bank (when at banker) |
| `/gb sort` | Sort your bags |
| `/gb config` | Open settings |

## Installation

### From CurseForge
1. Use CurseForge app to install
2. Or download and extract to: `World of Warcraft/_anniversary_/Interface/AddOns/`

### Manual Install
1. Download the latest release
2. Extract the folder to your AddOns directory
3. Restart WoW or type `/reload`
